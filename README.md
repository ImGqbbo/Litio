# Litio
How to use:
- Go info bin/Debug folder;
- Start Litio.exe;
- Insert your bot/account token (If bot token make dire you have inserted the 'Bot ' prefix before the token);
- Invite the bot with application.commands scope (for slash commands);
- Enjoy :')
